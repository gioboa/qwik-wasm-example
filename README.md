## Prerequisites

Install Rust and wasm-pack

```shell
curl https://sh.rustup.rs -sSf | sh
curl https://rustwasm.github.io/wasm-pack/installer/init.sh -sSf | sh
```

## Installation and run the server

```shell
pnpm install
```

```shell
pnpm dev
```
